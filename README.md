# El País Opinion Scraper Project

## Project Description
This project automates web scraping from the El País news website using Selenium automation framework.

The system extracts the first five articles from the Opinion section, downloads article images, translates Spanish titles into English using API integration, and performs repeated word analysis. It also supports parallel cross-browser testing using BrowserStack cloud platform.

## Features
- Scrapes first 5 articles from El País Opinion section  
- Downloads article cover images  
- Translates Spanish titles to English  
- Performs word frequency analysis  
- Executes parallel browser testing  

## How to Run

python main.py
